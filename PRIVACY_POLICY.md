# Privacy Policy

Forms Inline Image Preview does not collect, store, share, or sell personal information.

## What the extension does

The extension runs on Microsoft Forms pages and converts image-upload file links into inline image previews so responses are easier to review and print.

## Data handling

- No analytics
- No tracking
- No advertising
- No external data transfer
- No remote code
- No account creation

The extension processes page content locally inside the browser only to identify image links and display them inline.

## Third parties

The extension does not send user data to the developer or to third-party services.
